数据结构c语言描述六-数组

发表于 2017-12-17 | 分类于 数据结构

数组是数据结构中最基本的结构类型(再有些编程语言中并没有数组，比如python使用list替代数组，但还是有相近的地方,这里专门指c语言中的数组)，用于存储同一类型数据的数据结构

数组的顺序存储

数据一旦定义，其结构和元素间的关系不再发生改变，因此，采用顺序存储结构表示数组是比较合适的。优于地址的存储单元是一维的结构，而有的数组是多维的结构，使用一维的结构存储多维数组中的元素时就存在次序问题。针对二维数组的存放有两种方式：1.按行向量排列;2.按列向量排列

设有一二维数组A是: A[c1….d1,c2….d2]，其中(c1,c2)表示起始位置，(d1,d2)表示终点位置(这里的二维数据不一定是以(0,0)下标为起始位置的)，那么二维数组A有 d1-c1+1行，有d2-c2+1列。现求A[x][y]的存储位置(c1<=x<=d1,c2<=y<=d2)，
采用行优先存储可以计算得

LOC(x,y) = LOC(c1,c2) + ((d2 - c2 + 1) * (x - c1) + (y - c2)) * S
简单描述一下就是:数组的起始位置 + (x的前一行到起始位置的总行数 * 每一行的元素总数 + 要求位置到当前行开始位置的元素个数) * 每个元素占有的存储空间

采用列优先存储可以计算得

LOC(x,y) = LOC(c1,c2) + ((d1 - c1 + 1) * (y - c2) + (x - c1)) * S
简单描述一下就是:数组的起始位置 + (要求位置到当前行开始位置的元素个数 * 每一列的元素总数 + x的前一行到起始位置的总行数) * 每个元素占有的存储空间

数组的应用

对称矩阵的压缩存储

对称矩阵中的元素基于矩阵对角线相同，那么几乎有一半左右的元素是相同的，为了节省存储空间，可以将一对相同的元素存储在同一位置。

假设有一nn对称矩阵，原本需要的存储空间为n^2，使用对称矩阵可以将存储空间缩小为 n(n-1)/2，推导如下:

对称矩阵的第一行有一个元素，第二行有两个元素，第三行有三个元素，以此类推，第n行有n个元素，可以轻易计算出对称矩阵的存储空间为 n*(n-1) /2

将n*n的二维数组转换为一维数组那么就存在一个二维数组和一维数组之间的映射关系，需要给定一个二维数组的下标计算出在一维数组中存储值的下标，如果要求x行y列的对应存储值的下标，当x>=y时，可以先计算从第一行到x-1行的所有个数再加上y可以对应下标(这里的下标使用1开始计算，如果使用0开始计数，那么还要减1)，当x=y时计算。

int getIndex(int x,int y){
    if(y > x)
        return (y * (y - 1)) / 2 + x - 1;
    return (x * (x - 1)) / 2 + y - 1;
}

三元矩阵的十字链表存储

十字链表的每个节点有5个属性，分别表示矩阵行，列，数据，以及同一列下边元素的地址，同一行右边元素的地址，
十字链表有两个数组，一个代表行数组，一个代表列数组，每个数组中的每个元素分别表示每一行每一列的起始元素坐标，如果数组指向空，那么表示这一行或者这一列没有任何元素。
类型定义如下:

typedef struct _Node{
    int x,y;              //数组的行列值
    int data;             //存储数据
    struct _Node* right; //右边位置地址
    struct _Node* down;   //下边元素地址
}Node;
typedef struct {
    Node* col;       //一个数组，存放有每一列第一个元素的地址，为空表示这一列没有元素
    Node* row;       //一个数组，存放有每一行第一个元素的地址，为空表示这一行没有元素
    int col_total;   //列总数，col数组的长度
    int row_total;   //行总数，row数组的长度
    int total;       //非0元素总个数
}CrossList;

在插入元素是，要分别前往十字链表行列数组，创建和销毁指针数据

int insertCrossList(CrossList* cList,Node* n){
  //行数组移动指针
    if(cList->row[n->x].right == NULL){
        cList->row[n->x].right = n;
        n->right = NULL;
    }else{
        Node* ptr = cList->row[n->x].right;
        if(ptr->y == n->y){
            ptr->data = n->data;
        }else if(ptr->y > n->y){
            cList->row[n->x].right = n;
            n->right = ptr;
        }else{
            while(ptr->right != NULL){
                if(ptr->right->y == n->y){
                    ptr->right->data = n->data;
                    break;
                }else if(ptr->right->y > n->y){
                    n->right = ptr->right;
                    ptr->right = n;
                    break;
                }
                ptr = ptr->right;
            }
            if(ptr->right == NULL && ptr->y < n->y){
                ptr->right = n;
            }
        }
    }
  //列数组移动指针
    if(cList->col[n->y].down == NULL){
        cList->col[n->y].down = n;
        n->down = NULL;
    }else{
        Node* ptr = cList->col[n->y].down;
        if(ptr->x == n->x){
            ptr->data = n->data;
        }else if(ptr->x > n->x){
            cList->col[n->x].down = n;
            n->down = ptr;
        }else{
            while(ptr->down != NULL){
                if(ptr->down->x == n->x){
                    ptr->down->data = n->data;
                    break;
                }else if(ptr->down->x > n->x){
                    n->down = ptr->down;
                    ptr->down = n;
                    break;
                }
                ptr = ptr->down;
            }
            if(ptr->down == NULL && ptr->x < n->x){
                ptr->down = n;
            }
        }
    }
    return 1;
}
