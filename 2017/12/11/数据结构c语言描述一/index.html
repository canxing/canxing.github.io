数据结构c语言描述一

发表于 2017-12-11 | 分类于 数据结构 |

临近考试，对数据结构课程进行复习，参考资料《算法导论》。《数据结构》课程以数据结构为考查重点，所以不会涉及太多的算法方面的知识，使用c语言进行描述

数据结构

数据结构是一种存储和组织数据的方式，旨在便于访问和修改
数据结构的逻辑结构
    描述数据元素之间的逻辑关系，也可以看作是从具体问题抽象出来的模型，描述操作对象之间的关系，他与数据的存储方式、位置无关

    |-集合结构
    |-线性结构
    |-树形结构
    |-图形结构

数据结构的存储结构
    |-顺序存储：借助存储器中的位置来表示数据元素之间的关系
    |-链式存储：对逻辑上相邻的元素不要求求其物理位置相邻
    |-索引存储：建立附加索引表示标识节点地址

算法

算法要解决的问题：
    1.存在许多候选解，但绝大多数候选解都没有解决手头问题
    2.存在实际应用

特性：有穷性，确定性，可行性，输入，输出
目标：正确性，可读性，健壮性，效率与低存储量需求
算法复杂度
    |-时间复杂度
        |-算法事后统计方法
        |-事前分析估算方法
    |-空间复杂度

插入排序算法（伪代码）时间复杂度计算

INSERTION-SORT(A)

                                  每一行执行效率        执行次数

for j = 2 to A.length                   c1                   n

    key = A[j]                           c2                   n - 1

    i = j - 1                           c3                   n - 1

    while i > 0 and A[i] > key           c4                 Σ(2..n)tj    

        A[i + 1] = A[i]                   c5                 Σ(2..n)(tj-1)

        i = i - 1                       c6                 Σ(2..n)(tj-1)

    A[i + 1] key                       c7                   n - 1

tj表示while语句执行的次数

运行时间T(n) = c1n + c2(n-1) + c3(n-1) + c4Σ(2..n)tj + c5Σ(2..n)(tj-1) + c6Σ(2..n)(tj-1) + c7*(n-1)

最佳运行时间为tj = 1时

T(n) = c1n + c2(n-1) + c3(n-1) + c4Σ(2..n)tj + c7*(n-1)

可以表示为T(n) = a * n + b是线性结构

最坏运行时间

T(n) = c1n + c2(n-1) + c3(n-1) + c4(n-1)(n+2)/2 + c5(n-1)n/2 + c5(n-1)n/2 + c7*(n-1)

可以表示为T(n) = an2 + bn + c是二次函数
