AJAX是什么

AJAX = Asynchronous JavaScript and XML（异步的 JavaScript 和 XML）。
AJAX可以在页面不刷新的情况更新局部页面，也可以在页面加载过程中先加载重要部分在加载不那么重要的部分，提高页面加载速度
