python3-03-字符串

发表于 2018-04-30 | 更新于: 2018-05-01 | 分类于 python

字符串的构造方式

python将两个单引号，双引号，三引号（三个连续的单引号或双引号）扩起的内容当作字符串，一些字符串的形式如下： 

>>> print('afa1324')
afa1324
>>> print(" fa14319")
 fa14319
>>> print("""fadsf
... 1341
... fa""")
fadsf
1341
fa
>>> print('''adsf193249
... fdasj''')
adsf193249
fdasj

在单引号构造的字符串中可以包含双引号，双引号构造的字符串中可以包含单引号，如果想在单引号或双引号中使用单引号或双引号字符，那么就需要使用转义

转义

对于使用单引号构造的字符串中想要使用单引号或者在使用双引号构造的字符串中使用双引号，可以使用反斜杠 \ 对引号进行转义

>>> print('let\'s go!')
let's go!

当然也可以在双引号构造的字符串中直接使用单引号

>>> print("let's go!")
let's go!

如果想在字符串中打印反斜杠，可以使用反斜杠来对反斜杠进行转义

>>> print('a\\b')
a\b

基本操作

字符串属于序列的一种，字符串支持所有序列的操作

索引

>>> str = '123456789'
>>> str[0]
'1'
>>> str[5]
'6'
>>> str[-1]
'9'
>>> str[-8]
'2'

字符串可以通过索引获取值，但是不能通过索引来赋值

>>> str[1] = 'a'
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: 'str' object does not support item assignment

和序列一样，索引超过范围或报错

>>> str[-12]
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: string index out of range
>>> str[10]
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: string index out of range

切片

字符串也支持切片操作

>>> str[1:8]
'2345678'
>>> str[3:-3]
'456'
>>> str[1:8:2]
'2468'

相加

两个字符串可以通过加号合成一个新的字符串

>>> str + str1
'123456789abcdefghijklmn'

相乘

字符串可以和整数x相乘，将原本字符串的内容复制x份合成一个新的字符串

>>> 7 * str
'123456789123456789123456789123456789123456789123456789123456789'

成员资格

字符串支持使用运算符in来判断指定字符串是否在字符串中

>>> str
'123456789'
>>> 'a' in str
False
>>> '1' in str
True
>>> '123' in str
True

字符串方法

字符串含有大量的方法这里只给出部分方法

center

方法center可以在字符串两边填充字符（默认是空格）使字符串居中

>>> 'Tom'.center(10)
'   Tom    '
>>> 'Tom'.center(10, '-')
'---Tom----'

相似方法：ljust, rjust, zfill

>>> 'Tom'.ljust(10, '-')
'Tom-------'
>>> 'Tom'.ljust(1, '-')
'Tom'
>>> 'Tom'.rjust(10, '-')
'-------Tom'
>>> 'Tom'.zfill(10)
'0000000Tom'
>>> '+Tom'.zfill(10)
'+000000Tom'

find

方法find用于在字符串中查找字串，如果找到返回字串的第一个字符索引，否则返回-1，可以指定要查找的范围

>>> a = 'tom jerry tom jerry'
>>> a.find('tom')
0
>>> a.find('tom', 1)
10
>>> a.find('tom', 1, 4)
-1

相似方法：rfind, index, rindex, count, startswith, endswith

join

方法join用于合并序列元素，序列中的元素只能是字符串

>>> s = '-'
>>> a = ('1', '2', '3', '4', '5')
>>> s.join(a)
'1-2-3-4-5'
>>> s = '+'
>>> s.join(a)
'1+2+3+4+5'

与join方法相反的方法split用于分割字符串

>>> s
'1+2+3+4+5'
>>> s.split('+')
['1', '2', '3', '4', '5']

lower

方法lower返回字符串的小写版本

>>> s = s + 'ABCDEFG'
>>> s
'1+2+3+4+5ABCDEFG'
>>> s.lower()
'1+2+3+4+5abcdefg'

相似方法：upper，title，swapcase，casefold，capitalize
判断方法：islower，istitle，isupper，translate

replace

方法replace将指定字串替换为另一字符串，并返回替换后的结果

>>> s = 'tom and jerry'
>>> s.replace('tom', 'ton')
'ton and jerry'

strip

方法strip将字符串开头和结尾指定的字符删除默认是空白，并返回删除后的结果

>>> s = '     abcd efg    hijk       '
>>> s.strip()
'abcd efg    hijk'
>>> s = '*****abcdef   jgij********'
>>> s.strip('*')
'abcdef   jgij'

相似方法：lstrip，rstrip

判断字符串是否满足某种条件

isalnum 检查字符串中的字符是否是字母或数字
isalpha 检查字符串中的字符是否是字母
isdecima 检查字符串中的字符是否是十进制数
isdigit 检查字符串中的字符是否是数字
isidentifie 检查字符串中的字符是否可用作Python标识符
islower 检查字符串中的字母是否都是小写
isnumeri 检查字符串中的字符是否都是数字字符
isprintable 检查字符串中的字符是否都是可打印的
isspace 检查字符串中的字符是否都是空白字符
istitle 检查字符串中的字符位于非字母后面的字母都是大写的，且其他所有字母都是小写的
isupper 检查字符串中的字母是否都是大写的

格式化字符串

字符串提供一种简单方式可以将动态的值嵌入到字符串中，就像C语言中的printf函数一样，使用占位符占据位置，然后将值填充进去

python2 的格式化字符串

>>> s = 'hello %s'
>>> s % 'tom'
'hello tom'

python2使用和%将指定的格式来占位，然后填充，这里以python3为主，使用新方式格式化字符串，不多展开

python3 的格式化字符串

python3使用字符串的format方法来进行格式化，一个简单的格式化字符串如下

>>> s = 'hello {}'
>>> s.format('tom')
'hello tom'

在格式化字符串中可以自由选择以下部分组成:
字段名:索引或标识符
转换标志:跟在叹号(!)后面的单个字符，有r(表示repr)，s(表示str)，a(表示ascii)，默认就好
格式说明符:可以详细的指定要最终显示的格式，包括类型，字段宽度，精度，各种对齐方式等
