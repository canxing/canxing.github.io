python3-04-字典

发表于 2018-05-09 | 更新于: 2018-05-09 | 分类于 python

字典时python内置的一种通过名称来访问各个值的数据结构，其值不按顺序排列，而是存储在键下。

简单使用

存储

字典通常以以下方式表示

>>> a = {'a':123, 'b':456, 42:'afa', (123,34):'abcdef'}
>>> a
{42: 'afa', (123, 34): 'abcdef', 'a': 123, 'b': 456}

访问

通过存储的键来访问值

>>> a[42]
'afa'
>>> a[(123,34)]
'abcdef'
>>> a['a']
123
>>> a['b']
456

函数dict

出了通过以上方式构造字典还可以通过dict函数构造字典
函数dict的构造方式如下: 

dict() -> 创建一个空字典
    >>> a = dict()
    >>> a
    {}
dict(mapping) -> 从键值序列中创建字典，一个键值对就好象(key,value)
dict(iterable) -> 从一个可迭代类型中按照其中的键值对构建字典
    d = {}
    for k, v in iterable:
        d[k] = v
dict(**kwargs) -> new dictionary initialized with the name=value pairs
    in the keyword argument list.  For example:  dict(one=1, two=2)

基本操作

返回字典的键值对个数

>>> len({'a':1, 'b':2, 'c':3})
3

返回与键相关的值并且可以通过键赋新值

>>> a = {'a':1, 'b':2, 'c':3}
>>> a['a']
1
>>> a['a'] = 132
>>> a
{'c': 3, 'a': 132, 'b': 2}

删除指定键的键值对

>>> del a['b']
>>> a
{'c': 3, 'a': 132}

检查字典中是否包含指定键的键值对

>>> 'c' in a
True
>>> 'd' in a
False

字典方法

clear

方法clear用户清除所有字典中的键值对

>>> a = {'a':1, 'b':2, 'c':3}
>>> a.clear()
>>> a
{}

copy

方法copy返回一个新字典，新字典中的键值对与原来字典相同，但是修改新字典并不会影响原来的字典

>>> a = {'a':1, 'b':2, 'c':3}
>>> b = a.copy()
>>> b
{'a': 1, 'c': 3, 'b': 2}
>>> b['a'] = 2333
>>> b
{'a': 2333, 'c': 3, 'b': 2}
>>> a
{'b': 2, 'c': 3, 'a': 1}

fromkeys

方法fromkeys会将一个序列中的每个元素作为主键，指定值作为值，初始化为一个字典，这在对字典进行初始化赋值时很有用

>>> a.fromkeys(['a', 'b', 'c'])
{'b': None, 'c': None, 'a': None}
>>> a
{'b': 2, 'c': 3, 'a': 1}
>>> a.fromkeys('abcd')
{'b': None, 'c': None, 'a': None, 'd': None}

get

方法get返回键所对应的值，如果键不存在可以返回None或者其他指定值，和d[‘key’]相比使用get不容易抛出异常

>>> a
{'b': 2, 'c': 3, 'a': 1}
>>> a.get('a')
1
>>> a.get('d')
>>> a.get('d', 32)
32
>>> a['d']
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
KeyError: 'd'

items

方法items返回一个包含所有字典键值对的列表，其中每个元素为（key，value）形式

>>> a.items()
dict_items([('b', 2), ('c', 3), ('a', 1)])
>>> for k,v in a.items():
...     print(k,' ', v)
... 
b   2
c   3
a   1

keys

方法keys返回字典中所有键的列表

>>> a.keys()
dict_keys(['b', 'c', 'a'])
>>> for k in a.keys():
...     print(k)
... 
b
c
a

pop

方法pop用于获取对应键的值并将该键值对从字典中删除，如果键不存在会抛出异常

>>> a
{'b': 2, 'c': 3, 'a': 1}
>>> a.pop('a')
1
>>> a
{'b': 2, 'c': 3}
>>> a.pop('d')
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
KeyError: 'd'

popitem

方法popitem随即弹出字典中的一个键值对，如果字典为空则抛出异常

>>> a
{'b': 2, 'c': 3}
>>> a.popitem()
('b', 2)
>>> a
{'c': 3}

setdefault

方法setdefault用于为字典中的键设置默认值，如果该键没有值，那么获取时会返回默认值，如果该键有值，那么返回键对应的值

>>> a = {'a':1, 'b':2, 'c':3}
>>> a
{'b': 2, 'c': 3, 'a': 1}
>>> a.setdefault('d', None)
>>> a['d']
>>> a
{'b': 2, 'c': 3, 'a': 1, 'd': None}
>>> a.get('d')

values

方法values用于返回字典中所有值组成的列表

>>> a
{'b': 2, 'c': 3, 'a': 1, 'd': None}
>>> a.values()
dict_values([2, 3, 1, None])
