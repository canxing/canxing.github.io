  • 数据挖掘是从大量数据中挖掘有趣模式和知识的过程
  • 数据仓库是一个从多个数据源收集的信息存储仓库，存放在一致的模式下，并且通常驻留在单个站点上。数据仓库通过数据清理，数据变换，数据集成，数据装入和定期数据刷新来构造
  • 数据仓库用称做数据立方体的多维数据结构建模
  • 数据挖掘可以挖掘什么类型的模式
    1. 类/概念描述：特征化与区分
    2. 挖掘频繁模式，关联和相关性(支持度和置信度)
    3. 分类与回归
    4. 聚类分析(监督和非监督)

  • 机器学习
    1. 监督学习:基本上是分类的同义词。学习中的监督来自训练数据集中标记的实例
    2. 无监督学习:本质上是聚类的同义词。学习过程是无监督的，因为输入实例没有类标记
    3. 半监督学习:是一类机器学习技术，在学习模型时，它使用标记和未标记的实例。在一种方法中，标记的实例用来学习类模型，而未标记的实例用来进一步改进类边界
    4. 主动学习:是一种机器学习方法，它让用户在学习过程中扮演主动角色。主用学习方法可能要求用户对一个可能来自未标记的实例集或由学习程序合成的实例进行标记

  • 主成分分析或PCA的概念，自己去查
    • 是一种分析、简化数据集的技术。主成分分析经常用于减少数据集的维数，同时保持数据集中的对方差贡献最大的特征。这是通过保留低阶主成分，忽略高阶主成分做到的。这样低阶成分往往能够保留住数据的最重要方面。但是，这也不是一定的，要视具体应用而定。由于主成分分析依赖所给数据，所以数据的准确性对分析结果影响很大。
  • 联机事务处理(OLTP):主要任务是执行联机事务和查询处理
  • 联机分析处理(OLAP):可以用不同的格式组织和提供数据，以便满足不同用户的形形色色的需求
  • OLTP和OLAP的区别
    1. 用户和系统的面向性:OLTP是面向顾客的，用于办事员，客户和信息技术专业人员的事务和查询处理。OLAP是面向市场的，用于知识工人的数据分析
    2. 数据内容:OLTP系统管理当前数据
    3. 数据库设计:通常OLTP系统采用实体-联系（ER)数据模型和面向应用的数据库设计。而OLAP系统通常采用星型或雪花模型和面向主题的数据库设计
    4. 视图:OLTP系统主要关注一个企业或部门内部的当前数据，而不涉及历史数据或不同单位的数据。而OLAP系统还处理来自不同单位的信息，以及有多个数据库集成的信息
    5. 访问模式:OLTP系统的访问主要由短的原子事务组成。而OLAP系统的访问大部分是只读操作
  • 支持度和置信度 P158
    • 支持度: 计算同时包含A和B的元素个数和数据集中的百分比
  • 闭的 P160
    • 如果不存在真超项集Y使得Y与X在D中具有相同的支持度计数
  • Apriori算法 P160 代码 P163
    • 算法步骤:
    1. 返回单个元素在数据集中的支持度大于最小支持度的元素，作为候选项集
    2. 关联候选项集，生成新的项集，并计算新项集在数据集中的支持度，返回所有大于最小支持度的项集，作为候选项集
    3. 进行第二步，直到候选项集为空，返回最后的候选项集，这就是频繁项集

    

    #收集数据集中的单个元素
def createData(D) :
    retList = []
    for d in D:
        for i in d:
            if i not in retList :
                retList.append(i)
    return retList
# 关联频繁相集
def aprioriGen(Lk, k) :
    retList = []
    lenLk = len(Lk)
    for i in range(lenLk) :
        for j in range(i + 1, lenLk):
            L1 = Lk[i]
            L2 = Lk[j]
            if type(L1) == int :
                retList.append([L1, L2])
            else :
                L1 = list(L1[:k - 2])
                L2 = list(L2[:k - 2])
                L1.sort()
                L2.sort()
                if(L1 == L2):
                    retList.append(list(set(Lk[i]).union(set(Lk[j]))))
    return retList
# 主算法
def Apriori(D, minSup):
    L1 = scanfD(D, createData(D), minSup)
    L = [L1]
    k = 2
    while len(L[k - 2]) > 0:
        print(L)
        Ck = aprioriGen(L[k - 2], k)
        lk = scanfD(D, Ck, minSup)
        L.append(lk)
        k += 1
    return L[-2]
# 返回Ck中对于D大于最小支持度的相集
def scanfD(D, Ck, minSup): 
    retList = []
    ssCnt = {}
    numItems = float(len(D))
    for d in D:
        for c in Ck :
            if inlist(c, d) :
                if type(c) == list :
                    ssCnt[tuple(c)] = ssCnt.get(tuple(c), 0) + 1
                else:
                    ssCnt[c] = ssCnt.get(c, 0) + 1
    for key, value in ssCnt.items() :
        support = value / numItems
        if support > minSup :
            retList.append(key)
    return retList
# 判断一个整型或者一个列表是否在另一个列表中
def inlist(lst1, lst2) :
    if type(lst1) == list :
        for l in lst1:
            if l not in lst2 :
                return False
    else :
        return lst1 in lst2
    return True

  • KNN算法分类
    •  KNN算法步骤：  
       
    1.  计算输入点与所有点之间的距离
      2.  
    2.  找出前N个距离最近的点，N为自定义数量，小于等于已有点的数量
      3.  
    3.  统计最近的N个点的标签数量，数量最多的标签就是输入点的标签，也就是输入点的类型
      4.  
     
    


        def KNN(test, dataset, label, k) :
    

    # 计算数据的数量
size = dataset.shape[0]
# 计算欧式距离
diff = tile(test, (size, 1)) - group
sqdiff = diff ** 2
squareDist = sum(sqdiff, axis = 1)
dist = squareDist ** 0.5
# 将下标按照下标对应的值按照从小到大的顺序排列 
sortedDisIndex = argsort(dist)
print(sortedDisIndex)
classcount = {}
# 取距离最近的前几个，统计这几个的标签和对应的数量
for i in range(k):
    voteLabel = label[sortedDisIndex[i]]
    classcount[voteLabel] = classcount.get(voteLabel, 0) + 1
maxcount = 0
# 找出标签数量最多的标签，这个就是传入数据的标签

    if value > maxcount :

  • K-Means算法
    •  
       K-Means算法步骤: 
    1. 随便找K个点作为开始簇点，K为自定义数量，表示一堆数据中要有几个中心点，输入点会属于K的簇点中的一个
      2.  
    2. 依次遍历所有点，计算每个点与簇点之间的距离，找到和点最近的簇点，然后该簇点的簇群
      3.  
    3. 按照每个簇群中的点，重新计算簇点，计算公式为：该簇群中的所有点的横坐标的和除以该簇群中点的数量作为新簇点的横坐标，该簇群中所有点的纵坐标的和除以该簇群中点的数量作为新簇点的纵坐标
      4.  
    4. 跳转到第二步，直到簇点的位置不再改变（由于计算原因，计算结果会有误差，计算到一个足够小的值就可以停止了）
      5.  
    5. 计算输入点和所有簇点之间的距离，其中距离最近的簇点就是输入点所属的分类
      6.

    # 迭代簇点
def kMeansIter(cup, dataset) :
    dataSize = dataset.shape[0]
    size = len(cup)
    #dataSize -= size
    while True :
        groups = []
        for i in range(size) :
            groups.append([])
        for i in range(dataSize):
            minDis = 100000000 
            c = dataset[i]
            minIndex = -1;
            for j in range(size) :
                if minDis > distance(c, cup[j]) :
                    minDis = distance(c, cup[j])
                    minIndex = j
            groups[minIndex].append(c)
        newCup = []
        for group in groups :
            groupSize = len(group)
            sumX = 0
            sumY = 0
            for data in group :
                sumX += data[0]
                sumY += data[1]
            newX = sumX / groupSize
            newY = sumY / groupSize
            newCup.append(array([newX, newY]))
        isEquals = True
        for i in range(size):
            if distance(cup[i], newCup[i]) > 1:
                isEquals = False
                break
        if isEquals :
            break
        cup = newCup
    return cup, groups
